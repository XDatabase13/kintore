リクライニングスクワットマシンのやり方・効果Lying Machine Squat

主に効く: 大腿四頭筋 +臀筋 +ハムストリング マシン 中級者向け 定番種目

リクライニングスクワットマシン（Lying Machine Squat）は、シートに仰向けに近い姿勢で寝て行う、太もも前面にある大腿四頭筋を重点的に鍛えるためのマシン種目です。マシンが動きを安定させてくれるため、バランスを取るのが苦手な方でも比較的安心して重い重量を扱えます。補助的に、お尻の筋肉である臀筋や太もも裏のハムストリングも刺激される、中級者向けのトレーニングです。

鍛えられる筋肉

主に効く部位
大腿四頭筋
補助的に効く部位
臀筋、ハムストリング

主に、太ももの前面にある大腿四頭筋（だいたいしとうきん）に集中的に刺激を与えます。この筋肉は、膝を伸ばす動作の主役となる部位です。また、足を押し出す動作の際に、お尻の筋肉である臀筋（でんきん）や、太ももの裏側にあるハムストリングも補助的に使われます。リクライニング姿勢で行うことで、特に膝関節の動きにフォーカスしやすく、大腿四頭筋への強い収縮感を得やすいのが特徴です。

基本的なやり方

  1. マシンのシートに仰向けに近い姿勢で横たわり、肩パッドに肩をしっかりと押し付け、体を安定させます。
  2. 足裏全体をフットプレートに置き、肩幅程度に開きます。つま先は少し外側に向けるのが一般的ですが、ご自身が最も力を入れやすい角度を見つけましょう。
  3. 膝を少し曲げた状態で、安全バー（セーフティレバー）を解除し、マシンが自由に動く状態にします。
  4. 息を吸いながら、ゆっくりと膝を曲げて足を下ろしていきます。太ももとふくらはぎが90度くらいになるまで深く下ろすことを意識しましょう。
  5. 息を吐きながら、足裏全体でフットプレートを力強く押し返し、最初の位置まで戻ります。膝を完全に伸ばしきらず、少しゆとりを持たせると筋肉への刺激を保てます。
  6. 目標回数を行ったら、再度安全バーを元の位置に戻してマシンをロックし、ゆっくりとマシンの使用を終えます。

よくある質問

このリクライニングスクワットマシンは、通常のスクワットと何が違いますか？

通常のバーベルスクワットは全身でバランスを取りながら行うのに対し、リクライニングスクワットマシンは体がマシンに固定されているため、バランスを取る必要がありません。これにより、背中や腰への負担を軽減しやすく、大腿四頭筋に集中的に刺激を与えやすいという利点があります。フォームも安定しやすいので、より重い重量を扱いやすくなる場合があります。
